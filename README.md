\# AI-Powered Color Recommender System



An intelligent web application that suggests personalized color palettes based on user inputs. This project utilizes machine learning algorithms and rule-based logic to determine the most suitable colors for individuals (e.g., based on seasonal analysis or personal features).



\## Features



\* \*\*AI-Driven Suggestions:\*\* Uses a trained model to predict and recommend colors.

\* \*\*Web Interface:\*\* User-friendly interface built with Flask and HTML5.

\* \*\*Dynamic Design:\*\* Responsive styling with CSS and JavaScript integration.

\* \*\*Rule-Based Logic:\*\* Special implementation for specific color rules (e.g., Purple/Green analysis).



\## Tech Stack



\* \*\*Backend:\*\* Python, Flask

\* \*\*Frontend:\*\* HTML5, CSS3, JavaScript

\* \*\*Data/AI:\*\* NumPy, Scikit-learn

\* \*\*Version Control:\*\* Git \& GitHub

\## Installation \& Usage



To run this project locally, follow these steps:



1\. Clone the repository:

&nbsp;  git clone https://github.com/irmakknkcu/AI\_ColorRecommender.git

&nbsp;  cd AI\_ColorRecommender



2\. Create a Virtual Environment (Optional but recommended):

&nbsp;  python -m venv venv

&nbsp;  # Windows:

&nbsp;  venv\\Scripts\\activate

&nbsp;  # Mac/Linux:

&nbsp;  source venv/bin/activate



3\. Install Dependencies:

&nbsp;  pip install -r requirements.txt



4\. Run the Application:

&nbsp;  python app.py



5\. Open in Browser:

&nbsp;  Go to http://127.0.0.1:5000 to view the app.
